

<h1 align="center">Call Center</h1>
呼叫中心前台管理系统，提供技能组管理，IVR可视化管理，号码管理前台功能，可与FS后台服务进行结合使用
<div align="center">
## 部署运行
```bash
$ mkdir <your-project-name>
$ cd <your-project-name>
$ yarn create umi  # or npm create umi

$ npm install
$ npm start         # visit http://localhost:8000
```
